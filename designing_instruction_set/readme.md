### Designing an instruction set

- Antarmuka antara software dan hardware
- Aturan bahasa agar CPU tau (Instruction Set Architecture) ISA bahasa antara software dan hardware
- Contoh instruksi:
  - Add R1, R2, R3 -> jumlahkan R2 dan R3 simpan di R1
  - Load R1, [addr] -> ambil data dari alamat -> simpan di R1
  - Jump [label] -> lompat ke instruksi tertentu
