### CMOS (Complementary Metal-Oxide-Semiconductor)

- Bagaimana transistor membuat logika digital
- CMOS adalah teknologi untuk membuat gerbang logika digital dari transistor.
- Semua true/false, 0/1, if/else akan jadi arus listrik di transistor.
- Alurnya adalah, CMOS -> gerbang logika -> ALU -> CPU -> sistem -> program
- Semua gerbang logika seperti OR, AND, NOT, NAND, NOR, XOR. Semuanya diwujudkan secara fisik oleh CMOS.
