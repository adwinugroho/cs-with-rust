### Finite State Machines (FSM)

- Model komputasi berbasis state dan transisi
- Jembatan *hardware* ke *software*
- Model yang terdiri dari:
1. Kumpulan state
2. Transisi antar state berdasarkan input
3. Output (tergantung model Mealy atau Moore)
Contoh in real life:
- Lampu merah (Moore model, output tidak peduli input)
- ATM Machine
- CPU Control Unit = Fetch -> Decode -> Execute -> Write Back -> Ulang lagiii
- Lift (Mealy, output tergantung pada input)
