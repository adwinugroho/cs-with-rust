### Basic Information

Apa itu informasi?
- Representasi data dalam bentuk bit (0/1).
- Ukuran: bit, byte, word.
- Entropy, Shannon’s information theory.

Contoh: 0xFF, 1 << 3, binary.Read.

- Informasi itu bukan sekadar kata atau angka — tapi sesuatu yang mengurangi ketidakpastian.
- Semakin tak terduga suatu kejadian, semakin banyak informasi yang dibawanya.
- 1 Byte -> 8 Bit
